# Components Delta Skills

Este documento descreve apenas o que muda no uso de Skills.

## Onde consultar

1. [../Skills/README.md](../../Skills/README.md)
2. [../Skills/README.md](../../Skills/README.md)

## Quando usar

1. Para orientar implementação, testes e revisão técnica.
2. Para carregar contexto especializado por task.
3. Para guiar agentes por especialidade.

## Regras específicas

1. Skill não substitui workflow.
2. Skill não substitui Quality Gate.
3. Skill deve apontar para artefatos oficiais em Docs.
