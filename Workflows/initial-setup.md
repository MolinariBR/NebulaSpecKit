# Workflow: Setup Inicial do Projeto

## Gatilho

## Sequência recomendada

1. Bootstrap estrutural completo.
2. User Stories.
3. Contratos.
4. UI/UX.
5. Fluxo.
6. Prototype (quando aplicável).
7. Integração (quando aplicável).
8. Scripts.
9. Testes.
10. Deploy.
11. Planejamento e controle.
12. Quality Gate.
