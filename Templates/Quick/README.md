# Templates Quick

Template set resumido para execução rápida com rastreabilidade mínima.

## Objetivo

Executar documentação com baixa burocracia mantendo governança mínima.

## Regra de uso

1. Arquivos em `Templates/Quick/` são modelos.
2. Artefatos oficiais do projeto devem ser salvos em `Docs/` com os mesmos nomes canônicos.
3. Protótipos HTML devem ser salvos em `Docs/Prototype/`.

## Quando usar

1. Escopo pequeno e claro.
2. Risco de ambiguidade baixo.
3. Necessidade de velocidade com rastreabilidade mínima.

## Regras operacionais

1. Use arquivos de `Templates/Quick/` apenas como modelo.
2. Mantenha consistência com fases e milestones definidos no plano.
3. Registre progresso no `Docs/control.md` e evidências no Quality Gate.
4. Se houver ambiguidade, migrar para Full na mesma task.

## Leitura recomendada

1. `README.md` deste diretório
2. `Templates/Full/README.md`
3. `GUIDE.md`
4. `Docs/README.md`

## Relação com o modo Full

1. Se surgir ambiguidade, migrar do Quick para Full na mesma task.
